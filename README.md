# Crop Recommendation System

## Overview
This project recommends suitable crops based on soil and environmental conditions using data analysis and machine learning.

## Dataset
- Nitrogen, Phosphorus, Potassium
- Temperature, Humidity, pH, Rainfall

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Workflow
1. Data loading and preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature scaling
4. Model training and evaluation

## Result
The system predicts the most suitable crop for given conditions.
